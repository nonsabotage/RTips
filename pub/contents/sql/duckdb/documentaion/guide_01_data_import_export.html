3 CSV Export

CSVファイルにテーブルから取得したデータを書き出すには、COPYステートメントを使う。

Code 
CREATE OR REPLACE TABLE new_tbl AS SELECT * FROM read_csv('./mtcars.csv');
Code 
dbReadTable(con, "new_tbl") |> head()
#>    mpg cyl disp  hp drat    wt  qsec vs am gear carb
#> 1 21.0   6  160 110 3.90 2.620 16.46  0  1    4    4
#> 2 21.0   6  160 110 3.90 2.875 17.02  0  1    4    4
#> 3 22.8   4  108  93 3.85 2.320 18.61  1  1    4    1
#> 4 21.4   6  258 110 3.08 3.215 19.44  1  0    3    1
#> 5 18.7   8  360 175 3.15 3.440 17.02  0  0    3    2
#> 6 18.1   6  225 105 2.76 3.460 20.22  1  0    3    1
Code 
COPY new_tbl to './mtcars_from_duckdb.csv' (HEADER, DELIMITER '|', OVERWRITE_OR_IGNORE)

もしくは次のようにクエリ結果を出力することができる。

#| connection: con
COPY (SELECT * FROM new_tbl) to './mtcars_from_duckdb.csv' (HEADER, DELIMITER '|', OVERWRITE_OR_IGNORE)

# CSV Import

まずは単純に読込をおこなう。


```{sql}
#| connection: con
SELECT * FROM read_csv('./mtcars.csv');
```

読込と合わせてテーブルの作成をおこなう。


```{sql}
#| connection: con
CREATE TABLE new_tbl AS SELECT * FROM read_csv('./mtcars.csv');
```


```{r}
dbListTables(con)
```


```{r}
dbReadTable(con, "new_tbl")
```

既存のテーブルにデータを追加する。

```{sql}
#| connection: con
INSERT INTO new_tbl SELECT * FROM read_csv('./mtcars.csv');
```


```{r}
dbGetQuery(con, "SELECT COUNT(1), COUNT(*), COUNT() FROM new_tbl;")
```


`COPY`コマンドを使うと既存のテーブルにCSVファイルからデータをロードすることが可能である。これは**追加**になっていることに注意する。

```{sql}
#| connection: con
COPY new_tbl FROM './mtcars.csv'
```

```{r}
dbGetQuery(con, "SELECT COUNT(1), COUNT(*), COUNT() FROM new_tbl;")
```

# CSV Export


CSVファイルにテーブルから取得したデータを書き出すには、`COPY`ステートメントを使う。

```{sql}
#| connection: con
CREATE OR REPLACE TABLE new_tbl AS SELECT * FROM read_csv('./mtcars.csv');
```

```{r}
dbReadTable(con, "new_tbl") |> head()
```


```{sql}
#| connection: con
COPY new_tbl to './mtcars_from_duckdb.csv' (HEADER, DELIMITER '|', OVERWRITE_OR_IGNORE)
```

もしくは次のようにクエリ結果を出力することができる。

```sql
#| connection: con
COPY (SELECT * FROM new_tbl) to './mtcars_from_duckdb.csv' (HEADER, DELIMITER '|', OVERWRITE_OR_IGNORE)
```

# JSON Export

出力される形はオブジェクトの配列でなくて、オブジェクトが行ごとに並んでいることに注意する。
この方式だとjsonliteでパースができない。


```{sql}
#| connection: con
COPY (SELECT * FROM tbl) TO './penguins_from_duckdb.json' (OVERWRITE_OR_IGNORE)
```

```{r}
print(jsonlite::stream_in(file('./penguins_from_duckdb.json')))
```


```{r}
writexl::write_xlsx(palmerpenguins::penguins, "penguins.xlsx")
```


```{sql}
#| connection: con
SELECT * FROM st_read('./penguins.xlsx', layer = 'Sheet1');
```

オプションも少し付けることが可能。

```{sql}
#| connection: con
SELECT *
FROM st_read(
    './penguins.xlsx',
    layer = 'Sheet1',
    open_options = ['FIELD_TYPES=STRING']
);
```

# Excel Export

```{sql}
#| connection: con

```


```{sql}
#| connection: con

```
