1 はじめに

  • Spatialiteの使い方です.
  • DBIではモジュールがロードできないので使うことができないです
  • sfでデータを書き込む用に使うなどが適当な気がします

2 使い方

2.1 基本

  • コネクションの設定
  • 事前に、シェルで.load mod_spatialiteなどを実行しておく
  • これらはRSQLiteからは実行することができない
  • sfを使うにしても先にデータベースのファイルは作成しておく必要がある
.load mod_spatialite
select InitSpatialMetadata();
select spatialite_version();

target_db <- file.path(cur_dir, "spatial.sqlite")
st_write(
  pref, 
  target_db, 
  layer = "pref", 
  delete_layer = TRUE, 
  driver = "SQLite"
)

```shell

```
