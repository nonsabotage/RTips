1 はじめに

  • sfはRでGISを扱うためのパッケージです。
  • sfはSimple Featuresの略で、空間データを扱うための標準的なデータモデルです。
  • sfは、Rのtidyverseとの親和性が高く、dplyrのような文法で空間データを扱うことができます。

2 Applied Example

2.1 IDW

データを作成する。

Code 
locations <- 
  tibble(
    longitude = c(10, 10.5, 11),
    latitude = c(10, 14.5, 18),
    measurement = c(100, 150, 120)
  ) |>
  st_as_sf(coords = c("longitude", "latitude"), crs = 4326)

region <- st_as_sf(data.frame(
  x = c(10, 10, 20, 20),
  y = c(10, 20, 20, 10)
), coords = c("x", "y"), crs = 4326)

逆距離補間をおこなう。

Code 
grid <- st_make_grid(region, what = "centers")
idw_result <- idw(formula = measurement ~ 1, locations, grid)

plot(

    idw_result,

3 Fucntions

3.1 st_make_grid

  • 空間データをグリッドに変換する関数です。
  • 以下のように、グリッドのセルの大きさを指定することができます。
Code 
# 例として、ある地域のポリゴンを定義
region <- st_as_sf(data.frame(
  x = c(10, 10, 20, 20),
  y = c(10, 20, 20, 10)
), coords = c("x", "y"), crs = 4326)
Code 
grid <- st_make_grid(region, cellsize = 1, what = "polygons")
grid
#> Geometry set for 100 features 
#> Geometry type: POLYGON
#> Dimension:     XY
#> Bounding box:  xmin: 10 ymin: 10 xmax: 20 ymax: 20
#> Geodetic CRS:  WGS 84
#> First 5 geometries:
#> POLYGON ((10 10, 11 10, 11 11, 10 11, 10 10))
#> POLYGON ((11 10, 12 10, 12 11, 11 11, 11 10))
#> POLYGON ((12 10, 13 10, 13 11, 12 11, 12 10))
#> POLYGON ((13 10, 14 10, 14 11, 13 11, 13 10))
#> POLYGON ((14 10, 15 10, 15 11, 14 11, 14 10))
Code 
plot(grid)
plot(region, add = TRUE)
# ポイントデータを作成することもできる
plot(
  st_make_grid(region, cellsize = 1, what = "centers"),
  col = "red", 
  add = TRUE
)
plot(
  st_make_grid(region, cellsize = 1, what = "corners"),
  col = "blue", 
  add = TRUE
)

セルサイズはXYで変更することもできる。

Code 
plot(grid)
plot(
  st_make_grid(region, cellsize = c(1, 2), what = "centers"),
  col = "red", 
  add = TRUE
)
