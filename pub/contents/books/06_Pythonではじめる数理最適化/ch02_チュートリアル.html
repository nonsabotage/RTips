1 はじめに

二次方程式、線形計画問題を例としたPythonによる数理最適化のチュートリアルである。 必要なライブラリはrequirements.txtに記載されている。

必要なデータはPyOptBook以下の各章のディレクトリに格納されている。

2 連立1次方程式の解

PuLPを使って連立1次方程式を解く。

1個120円のりんごと1個150円のなしをあわせて10個購入したら、代金は1440円であった。りんごとなしの個数を求めよ。

Code 
import pulp

# 問題の定義
problem = pulp.LpProblem('SLE', pulp.LpMaximize)

# 変数の定義
x = pulp.LpVariable('x', lowBound=0, cat='Continuous')
y = pulp.LpVariable('y', lowBound=0, cat='Continuous')

# 制約条件
problem += x + y == 10
problem += 120*x + 150*y == 1440

# 最適化
status = problem.solve()

# 結果の表示
print(f'りんごの個数: {x.value()}')
#> りんごの個数: 2.0
print(f'なしの個数: {y.value()}')
#> なしの個数: 8.0

3 線形計画問題

Code 
problem = pulp.LpProblem('LP', pulp.LpMaximize)

x = pulp.LpVariable('x', lowBound=0, cat='Continuous')
y = pulp.LpVariable('y', lowBound=0, cat='Continuous')

problem += 1 * x + 3 * y <= 30
problem += 2 * x + 1 * y <= 40
problem += x >= 0
problem += y >= 0
problem += x + 2 * y

problem.solve()
#> 1

# 解の状態について表示する
print(f"Status: {pulp.LpStatus[status]}")
#> Status: Optimal
print(f"x: {x.value()}, y: {y.value()}, Objective: {problem.objective.value()}")
#> x: 18.0, y: 4.0, Objective: 26.0

data_dir = Path("./unshare/PyOptBook/2.tutorial")
stock_df = pd.read_csv(data_dir / "stocks.csv")
