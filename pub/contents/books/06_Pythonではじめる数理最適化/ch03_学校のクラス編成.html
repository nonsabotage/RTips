1 はじめに

学校のクラス編成について、問題を定式化して解いてみる。

大事なことは結果を検証して、必要なモデリングが行えているのかを考察すること。

条件は次のとおりである。

  1. 全生徒をそれぞれ１つのクラスに割り当てる
  2. 各クラスには、３９人以上、４０人以下の生徒が所属する
  3. 各クラスには、男子生徒、女子生徒はそれぞれ２０人以下とする
  4. 各クラスの学力試験の平均点は学年平均点±10点とする
  5. 各クラスにリーダー気質の生徒を2人以上所属させる
  6. 特別支援が必要な生徒は各クラスに一人以下とさせる
  7. 特定ペアの生徒は同一暮らすに割り立てない

2 実装

まずは目的関数を決めずに条件を満たす解を求解する。 ダミー変数をLp.Binaryで定義するのがポイントと思われる。

Code 
import pandas as pd
import pulp
from pathlib import Path

data_dir = Path("unshare", "PyOptBook", "3.school")

# データの読み込み
s_df = pd.read_csv(data_dir / "students.csv")
s_pair_df = pd.read_csv(data_dir / "student_pairs.csv")

# 問題の定義
prob = pulp.LpProblem("school", pulp.LpMaximize)


# 前処理
S = s_df["student_id"].tolist()

C = list("ABCDEFGH")

SC = [(s, c) for s in S for c in C]

S_male   = [row.student_id for row in s_df.itertuples() if row.gender == 1]
S_female = [row.student_id for row in s_df.itertuples() if row.gender == 0]

S_leader = [row.student_id for row in s_df.itertuples() if row.leader_flag == 1]
S_support = [row.student_id for row in s_df.itertuples() if row.support_flag == 1]
SS = [(row.student_id1, row.student_id2) for row in s_pair_df.itertuples()]

score = {row.student_id:row.score for row in s_df.itertuples()}
score_mean = s_df["score"].mean()



# 変数の定義
x = pulp.LpVariable.dicts("x", SC, 0, 1, pulp.LpBinary)

# 制約条件1
for s in S:
    prob += pulp.lpSum(x[(s, c)] for c in C) == 1

# 制約条件2
for c in C:
    prob += pulp.lpSum(x[(s, c)] for s in S) >= 39
    prob += pulp.lpSum(x[(s, c)] for s in S) <= 40

# 制約条件3
for c in C:
    prob += pulp.lpSum(x[(s, c)] for s in S_male) <= 20
    prob += pulp.lpSum(x[(s, c)] for s in S_female) <= 20
    
# 制約条件4
for c in C:
    prob += pulp.lpSum(score[s] * x[(s, c)] for s in S) >= (score_mean - 10) * pulp.lpSum(x[(s, c)] for s in S)
    prob += pulp.lpSum(score[s] * x[(s, c)] for s in S) <= (score_mean + 10) * pulp.lpSum(x[(s, c)] for s in S)

# 制約条件5
for c in C:
    prob += pulp.lpSum(x[(s, c)] for s in S_leader) >= 2

# 制約条件6
for c in C:
    prob += pulp.lpSum(x[(s, c)] for s in S_support) <= 1

# 制約条件7
for s1, s2 in SS:
    for c in C:
        prob += x[(s1, c)] + x[(s2, c)] <= 1

# 目的関数
status = prob.solve()
print(f"Status: {pulp.LpStatus[status]}")
#> Status: Optimal

s_df["score_rank"] = s_df["score"].rank(ascending=False, method = "first")

# はじめに

学校のクラス編成について、問題を定式化して解いてみる。

大事なことは結果を検証して、必要なモデリングが行えているのかを考察すること。


条件は次のとおりである。

1. 全生徒をそれぞれ１つのクラスに割り当てる
2. 各クラスには、３９人以上、４０人以下の生徒が所属する
3. 各クラスには、男子生徒、女子生徒はそれぞれ２０人以下とする
4. 各クラスの学力試験の平均点は学年平均点±10点とする
5. 各クラスにリーダー気質の生徒を2人以上所属させる
6. 特別支援が必要な生徒は各クラスに一人以下とさせる
7. 特定ペアの生徒は同一暮らすに割り立てない

# 実装

まずは目的関数を決めずに条件を満たす解を求解する。
ダミー変数を`Lp.Binary`で定義するのがポイントと思われる。

```{python}
#| label: implementation
import pandas as pd
import pulp
from pathlib import Path

data_dir = Path("unshare", "PyOptBook", "3.school")

# データの読み込み
s_df = pd.read_csv(data_dir / "students.csv")
s_pair_df = pd.read_csv(data_dir / "student_pairs.csv")

# 問題の定義
prob = pulp.LpProblem("school", pulp.LpMaximize)


# 前処理
S = s_df["student_id"].tolist()

C = list("ABCDEFGH")

SC = [(s, c) for s in S for c in C]

S_male   = [row.student_id for row in s_df.itertuples() if row.gender == 1]
S_female = [row.student_id for row in s_df.itertuples() if row.gender == 0]

S_leader = [row.student_id for row in s_df.itertuples() if row.leader_flag == 1]
S_support = [row.student_id for row in s_df.itertuples() if row.support_flag == 1]
SS = [(row.student_id1, row.student_id2) for row in s_pair_df.itertuples()]

score = {row.student_id:row.score for row in s_df.itertuples()}
score_mean = s_df["score"].mean()



# 変数の定義
x = pulp.LpVariable.dicts("x", SC, 0, 1, pulp.LpBinary)

# 制約条件1
for s in S:
    prob += pulp.lpSum(x[(s, c)] for c in C) == 1

# 制約条件2
for c in C:
    prob += pulp.lpSum(x[(s, c)] for s in S) >= 39
    prob += pulp.lpSum(x[(s, c)] for s in S) <= 40

# 制約条件3
for c in C:
    prob += pulp.lpSum(x[(s, c)] for s in S_male) <= 20
    prob += pulp.lpSum(x[(s, c)] for s in S_female) <= 20
    
# 制約条件4
for c in C:
    prob += pulp.lpSum(score[s] * x[(s, c)] for s in S) >= (score_mean - 10) * pulp.lpSum(x[(s, c)] for s in S)
    prob += pulp.lpSum(score[s] * x[(s, c)] for s in S) <= (score_mean + 10) * pulp.lpSum(x[(s, c)] for s in S)

# 制約条件5
for c in C:
    prob += pulp.lpSum(x[(s, c)] for s in S_leader) >= 2

# 制約条件6
for c in C:
    prob += pulp.lpSum(x[(s, c)] for s in S_support) <= 1

# 制約条件7
for s1, s2 in SS:
    for c in C:
        prob += x[(s1, c)] + x[(s2, c)] <= 1

# 目的関数
status = prob.solve()
print(f"Status: {pulp.LpStatus[status]}")
```
