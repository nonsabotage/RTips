9 増減率と対数差分系列

時系列分析では増減率をそのまま扱うのではなく、対数差分系列で近似する。

\[ \Delta \log y_t = \log y_t - \log y_{t-1} \]

増減率の近似として対数差分系列を扱う。

\[ \Delta \log y_t \approx \frac{y_t - y_{t-1}}{y_{t-1}} \]

Code

10 自己相関係数とコレログラム

自己相関を可視化したものをコレログラムという。

自己相関係数は次の式で定義される。

\[ \rho_k = \frac{\gamma_k}{\gamma_0} \]

ここで、\(\gamma_k\)は自己共分散関数であり、次の式で定義される。

\[ \gamma_k = \frac{1}{n} \sum_{t=1}^{n-k} (y_t - \bar{y})(y_{t+k} - \bar{y}) \]

auto_covar_5が正しいのだけど、auto_covar_6と何が違うのかわからない。。。。。

pandasのSeries型演算していることで、積がインデックスに基づいて行われているためであった。。。

なので、.valuesをつけて、numpyのarray型に変換することで、正しい計算ができる.

np.dotは自動でpandasのSeries型をnumpyのarray型に変換してくれるので、そのまま使える。

Code 
def auto_covar(data, lag):
    N  = len(data)
    mu = data.mean()
    return np.sum((data[lag:] - mu) * (data[:N-lag] - mu)) / N

def auto_covar_2(data, lag):
    N  = len(data)
    mu = data.mean()
    return np.sum((data[lag:] - mu) * (data[:N-lag] - mu)) / (N + lag)

def auto_covar_3(data, lag):
    N  = len(data)
    mu = data.mean()
    return np.sum((data[lag:] - mu) * (data[:N-lag] - mu)) / (N - 1)

def auto_covar_4(data, lag):
    N  = len(data)
    x0 = data - data.mean()
    return np.dot(x0[(lag+1):], x0[:-(lag+1)]) / N

def auto_covar_5(data, lag):
    N  = len(data)
    xo = data - data.mean()
    return np.dot(xo[lag:], xo[:N-lag])/N

def auto_covar_6(data, lag):
    N  = len(data)
    xo = data - data.mean()
    return np.sum(xo[lag:] * xo[:N-lag])/N
