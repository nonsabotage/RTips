air_passengers = sm.datasets.get_rdataset("AirPassengers").data

air_passengers.index = pd.date_range(
    start='1949-01-01', 
    periods=len(air_passengers), 
    freq='MS'
)

train = air_passengers.loc['1949-01':'1957-12']
test = air_passengers.loc['1958-01':'1960-12']

4 SARIMAXモデルとSARIMA ERRORモデル

外生変数入りのSARIMAモデルをSARIMAXモデルと呼びます。また、SARIMAモデルの残差をモデル化したものをSARIMA ERRORモデルと呼びます。

ARAMAXモデルは、外生変数を含むARIMAモデルです。SARIMAXモデルは、外生変数を含むSARIMAモデルです。SARIMA ERRORモデルは、残差をモデル化したSARIMAモデルです。 数式では次のように表される

ARMAXモデルは次です。

\[ \phi(B)y_t = \beta x_t + \theta(B) \epsilon_t \]

ARMA Errorモデルは次です。

\[ Y_t = \phi_1 Y_{t-1} + \cdots + \phi_p Y_{t-p} + \theta_1 \epsilon_{t-1} + \cdots + \theta_q \epsilon_{t-q} + \epsilon_t + \beta_1 X_{1,t} + \cdots + \beta_k X_{k,t} + \epsilon_t \]
