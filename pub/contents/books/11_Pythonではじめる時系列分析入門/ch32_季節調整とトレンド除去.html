2 季節調整とトレンド除去の考え方

時系列データには、次のような要素が含まれることがあります。

  • Trend: 長期的な変動
  • Circulars: 周期があいまいな循環成分
  • Seasonality: 季節変動
  • Irregular: 不規則成分

これらの要因を加法型と乗法型に分けて考えることができます。

  • 加法型: 時系列データがトレンド、季節変動、不規則成分の和で表される場合
  • 乗法型: 時系列データがトレンド、季節変動、不規則成分の積で表される場合

\[ \begin{align*} y_t &= T_t + C_t+S_t + I_t \\ y_t &= T_t \times C_t \times S_t \times I_t \end{align*} \]

たとえば「加法型の季節調整」は、時系列データから季節変動を抽出することを意味します。

つまり、次の式を目指すことになる。

\[ y_t - S_t = T_t + C_t + I_t \]

3.2 季節成分

不規則西部は長期的な平均値をとることによって無視できると考える。すなわり、\(S_t+I_t\)の平均値を季節成分\(S_t\)として抽出する。

Code

seasonal_year = seasonal_year - np.mean(seasonal_year)

8 差分による季節調整とトレンド除去

12時点前との季節成分の差分を取ることで、季節成分を除去することができる。

Code 
# 12時点差分
transformer_diff_12 = Differencer(lags=12)

# 変換の実施
desea_diff = transformer_diff_12.fit_transform(air_passengers.to_period())

# 差分によるトレンド除去
transformer_diff_1 = Differencer(lags=1)

detrend_diff = transformer_diff_1.fit_transform(air_passengers.to_period())
