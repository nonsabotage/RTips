3 モデル選択の手続き

  1. 季節単位根検定の１種であるOCBS検定を用いて、季節差分の階数を決定する。
  2. 単位根検定の１種であるKPSS検定を用いて、通常の差分の階数を決定する。
  3. さまざまな次数でモデルを推定する
  4. 情報量規準の一種であるAICなどを用いてモデルの次数の候補を選択
  5. 残差の診断を行い、モデルの適合性を確認する
  6. テストデータを用いてモデルの予測性能を評価する

mermaidで上記を図示すると以下のようになる。

graph TD
    A(モデル選択開始) --> B{OCSB検定};
    B --> |単位根あり| C[1階差分を取る] --> B;
    B --> |単位根なし| D{KPSS検定};
    D --> |単位根あり| E[1階差分を取る] --> D;
    D --> |単位根なし| F[次数を変えてモデル推定] --> G[AICなどで次数を選択];
    G --> H[残差診断];
    H --> I[予測精度の評価];

4 単位根検定：DF検定とADF検定

\(d\)階差分をとることで定常になる過程のことを\(d\)次和分過程という。\(d\)次和分過程の\(d\)をその過程の階数という。単位根検定は、時系列データが単位根過程であるかどうかを検定する手法である。対象となるデータ系列が単位根をもつならば、差分をとり定常過程に変換します

外生変数を利用する場合は、SARIMAモデルを残差に対して適用するため、残差に対して単位根検定をおこなう。

Code 
ols_resid = smf.ols('np.log(sales) ~ discount', data=train).fit().resid
