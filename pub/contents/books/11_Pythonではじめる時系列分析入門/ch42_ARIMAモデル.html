2 AR 過程

\(p\)次の自己回帰過程（AutoRegressive process）は、次のように表される。

\[ X_t = \phi_1 X_{t-1} + \phi_2 X_{t-2} + \cdots + \phi_p X_{t-p} + \epsilon_t \]

2.1 シミュレーション

1次のAR過程で回帰係数が正の場合と負の場合を考える。

\[ \begin{align} y_t &= 0.8y_{t-1} + \epsilon_t\\ y_t &= -0.8y_{t-1} + \epsilon_t \end{align} \]

2.2 AR過程に従うデータの自己相関

\(\|\phi\|<1\)なのでどちらの過程でも収束する様子がわかる。また、負の場合には自己相関が振動していることがわかる。

Code 
fig, ax = plt.subplots(2, 1, figsize=(8, 4), tight_layout=True)

_ = sm.graphics.tsa.plot_acf(ar_data['posi'], lags=40, title='AR posi ACF', ax=ax[0])
_ = sm.graphics.tsa.plot_acf(ar_data['nega'], lags=40, title='AR nega ACF', ax=ax[1])

2.3 AR過程に従うデータの偏自己相関

偏自己相関とは\(k-1\)次のまでの自己相関の影響を排除したうえで、純粋な\(k\)次の自己相関を求めるものである。

Code 
fig, ax = plt.subplots(2, 1, figsize=(8, 4), tight_layout=True)

_ = sm.graphics.tsa.plot_pacf(ar_data['posi'], lags=40, title='AR posi ACF', ax=ax[0])
_ = sm.graphics.tsa.plot_pacf(ar_data['nega'], lags=40, title='AR nega ACF', ax=ax[1])

3 MA過程

\(1\)次の移動平均過程（Moving Average process）は、次のように表される。

\[ y_t = \mu + \theta_1 \epsilon_{t-1} + \epsilon_t \]

3.1 MA過程に従うデータの例

\[ \begin{align} y_t &= 0.8\epsilon_{t-1} + \epsilon_t\\ y_t &= -0.8\epsilon_{t-1} + \epsilon_t \end{align} \]

Code 
ma_data = pd.read_csv("./book-python-tsa-intro/book-data/4-2-2-ma-data.csv", index_col='date', parse_dates=True, dtype='float')
