1 傾向スコアによる層化解析法および重み付き法

ここまでに傾向スコアマッチングによりATTを推定できること， 仮定が満たされているならば共分散分析によりATEが推定できることも確認した． しかし，ATEを推定するための共分散分析の仮定を満たすことは現実的には難しい.

そこで登場するのが傾向スコアを用いた層化解析手法である.

1.1 傾向スコアによる層化解析法

まず層化抽出とは，母集団をいくつかのグループに分けて， それぞれのグループで無作為抽出する方法である． このときのグループは調べる共変量が似通った値になるように調整する． たとえば「県民性」を調べたいときには 日本全国からサンプリングするよりも４７都道府県別でサンプリングする方がよいのは 直感的に理解できる． 層別は「意味がある」特性が行うことが重要である．

層別サンプリングは交絡因子を使って層にわけることで， 交絡因子の影響を受けるするという考えであり， フィッシャーの三原則の「局所管理」にあたる作法である．

層ごとの推定値の情報をまとめたものを全体の推定値とする． \(K\)層に分けた場合のATEは次式で求められる.

\[ \begin{align} \hat{\tau}_{\text{ATE}}&=\sum_{k=1}^{K}\frac{n_k}{N}[\overline{Y_k}(1)-\overline{Y_k}(0)]\\ \text{var}(\hat{\tau}_{\text{ATE}})&=\sum_{k=1}^{K}\left(\frac{n_k}{N}\right)^2[\overline{Y_k}(1)-\overline{Y_k}(0)] \end{align} \]

交絡変数をなくそうとして多変量で層別にすると， パターンは膨大になるので，傾向スコアで層別するのがやはり基本となる．

1.3 傾向スコアによる重み付け法

調査標本における重み付けと似ている． 調査標本ではある個体がサンプリングされる確率が等確率にならない， つまり，ランダムサンプリングが実現できない現象である．

例えば男子6000人，女子4000人の学校で1000人を対象にしたアンケートを行うとき， 女子ロッカーに関することのため女子の人数を多くしたアンケートをしたいと考える． このとき，男子120人，女子880人をサンプリングすると， 男子は120/6000なので0.02, 女子は880/4000なので0.22の確率でサンプリングされる．

そこでサンプリングされた個人の意見はサンプリング確率の逆数， この場合だと男子は50人, 女子は4.8人分の価値があると解釈する． これが，重み付け法である．

1.3.1 Rによる方法

標本調査では抽出確率の逆数を使っていた． 傾向スコアは共変量\(X\)があたてられたとき，処置に割り付けられる確率という意味であることから， その逆数を重みとして使うことは自然である． これを逆重み付け法(IPW)とおいう.

無作為化実験から得られるであろう解析結果に一致するように， 処置群の個体に対して「傾向スコアの逆数」を乗じて， 統制群の個体に対して「１－傾向スコア」の逆数を乗じることで，データの重み付けを算出する． つまり出にくい方の値が出たときにはその価値を重視するというものとなる．

\[ w_i = \frac{T_i}{\hat{e_i}}+\frac{1-T_i}{1-\hat{e_i}} \]

上記の計算は傾向スコアを算出したのちに，そのスコアを重みとした 回帰分析を行うことで算出が可能である.

Code 
model1  <- glm(t1 ~ x1 + x2 + x3 + x4 + x5 + x6, data = data11, family = binomial(link = "logit"))
ps1     <- model1$fitted.values
weights <- data11$t1 / ps1 + (1 - data11$t1) / (1 - ps1)
Code 
model2 <- lm(y3 ~ t1, data = data11, weights = weights)
model3 <- lm(y3 ~ t1 + x1 + x2 + x3 + x4 + x5 + x6, data = data11, weights = weights)

直感的に理解できる．
層別は「意味がある」特性が行うことが重要である．

層別サンプリングは交絡因子を使って層にわけることで，
交絡因子の影響を受けるするという考えであり，
フィッシャーの三原則の「局所管理」にあたる作法である．

層ごとの推定値の情報をまとめたものを全体の推定値とする．
$K$層に分けた場合のATEは次式で求められる. 

$$
\begin{align}
\hat{\tau}_{\text{ATE}}&=\sum_{k=1}^{K}\frac{n_k}{N}[\overline{Y_k}(1)-\overline{Y_k}(0)]\\
\text{var}(\hat{\tau}_{\text{ATE}})&=\sum_{k=1}^{K}\left(\frac{n_k}{N}\right)^2[\overline{Y_k}(1)-\overline{Y_k}(0)]
\end{align}
$$

交絡変数をなくそうとして多変量で層別にすると，
パターンは膨大になるので，傾向スコアで層別するのがやはり基本となる．

```{r}
library(lmtest); library(sandwich);
fits <- 
    m.data2 |> 
    group_by(subclass) |> 
    summarise(
        fit = list(lm(y3 ~ t1 + x1 + x2 + x3 + x4 + x5 + x6, data = cur_data()))
    )
class_size <- 
    m.data2 |> 
    count(subclass, name = "N")
fit_stats <- 
    fits |> 
    mutate(coef = map(fit, ~ tidy(.x))) |> 
    select(!fit) |> 
    unnest(coef) |> 
    filter(term == "t1") 
ate <- 
    fit_stats|> 
    left_join(class_size, by = "subclass") |> 
    mutate( 
        psvar = std.error ^ 2
    ) |> 
    select(
        subclass, 
        psp = estimate, 
        psvar, 
        nps = N, 
        p.value
    )
    
ate
    

```

estimated <-

## 傾向スコアによる重み付け法

調査標本における重み付けと似ている．
調査標本ではある個体がサンプリングされる確率が等確率にならない，
つまり，ランダムサンプリングが実現できない現象である．

例えば男子6000人，女子4000人の学校で1000人を対象にしたアンケートを行うとき，
女子ロッカーに関することのため女子の人数を多くしたアンケートをしたいと考える．
このとき，男子120人，女子880人をサンプリングすると，
男子は120/6000なので0.02, 女子は880/4000なので0.22の確率でサンプリングされる．

そこでサンプリングされた個人の意見はサンプリング確率の逆数，
この場合だと男子は50人, 女子は4.8人分の価値があると解釈する．
これが，重み付け法である．

### Rによる方法

標本調査では抽出確率の逆数を使っていた．
傾向スコアは共変量$X$があたてられたとき，処置に割り付けられる確率という意味であることから，
その逆数を重みとして使うことは自然である．
これを**逆重み付け法(IPW)**とおいう. 

無作為化実験から得られるであろう解析結果に一致するように，
処置群の個体に対して「傾向スコアの逆数」を乗じて，
統制群の個体に対して「１－傾向スコア」の逆数を乗じることで，データの重み付けを算出する．
**つまり出にくい方の値が出たときにはその価値を重視する**というものとなる．


$$
w_i = \frac{T_i}{\hat{e_i}}+\frac{1-T_i}{1-\hat{e_i}}
$$


上記の計算は傾向スコアを算出したのちに，そのスコアを重みとした
回帰分析を行うことで算出が可能である. 

```{r}
model1  <- glm(t1 ~ x1 + x2 + x3 + x4 + x5 + x6, data = data11, family = binomial(link = "logit"))
ps1     <- model1$fitted.values
weights <- data11$t1 / ps1 + (1 - data11$t1) / (1 - ps1)
```


```{r}
model2 <- lm(y3 ~ t1, data = data11, weights = weights)
model3 <- lm(y3 ~ t1 + x1 + x2 + x3 + x4 + x5 + x6, data = data11, weights = weights)
```
