2 外生変数

\[ \begin{align} Y_i &= \beta_0 + \beta_1 X_i + \epsilon_i \\ i &= 1, ..., n \end{align} \] 上記のモデルを考えたとき\(X_iが外生変数である\)といった意味は、次式で表されるように誤差項と相関しない、 ということを意味する。

$$

[X_i_i] = (X_i, _i) = 0

$$

一般に個体\(i\)の要因\(X_i\)を分析者側がランダムに決定する場合, \(X_i\)は明らかに外生変数となる。

Code 
data("ToothGrowth")
ToothGrowth |> 
    lm(len ~ dose, data = _) |> 
    tidy()
#> # A tibble: 2 × 5
#>   term        estimate std.error statistic  p.value
#>   <chr>          <dbl>     <dbl>     <dbl>    <dbl>
#> 1 (Intercept)     7.42     1.26       5.89 2.06e- 7
#> 2 dose            9.76     0.953     10.3  1.23e-14

#　内生変数

外生変数の条件が満たされない次の状態を考える。

$$

[X_i_i] = (X_i, _i)

$$

内生変数はいいといして、どのようなケースで内生変数が生じやすいのかを考える。

2.1 欠落変数

もとのモデルは内生変数がない次の状態を考える。この状態であれば通常の回帰分析で推定することができる。

$$

Y_i = _0 + 1X{1i} + 2X{2i} + _i

$$

ここで\(X_2\)が観測できない状態を考える。

$$ \[\begin{align} Y_i &= \alpha_0 + \beta_1X_{1i} + \eta_i \\ \alpha_0 &= \beta_0 + \beta_2\textrm{E}[X_{2i}] \\ \eta_i &= \beta_2(X_{2i}-\textrm{E}[X_{2i}])+\epsilon_i \end{align}\] $$

単回帰モデルで正しく\(\beta\)を推定するためには、\(X_{i}\)が誤差項と外生的変数であることが求められる。 下記の結果をみると\(X_{1i}\)は外生的であるため欠落したままでは回帰分析は適していない

\[ \begin{align} \textrm{E}(X_{1i}) &= \textrm{E}\{X_{1i}[\beta_2(X_{2i}-\textrm{E}[X_{2i}])+\epsilon_i]\} \\ &=\beta_2\textrm{E}{X_{1i}(X_{2i}-\textrm{E}[X_{2i}])} + E[X_{1i}\epsilon_i]\\ &=\beta_2\textrm{Cov}(X_{1i}., X_{2i}) \end{align} \]

2.2 測定誤差

測定誤差があると、過小バイアスがかかる。ただし符号は変化しない。

Code 
n <- 200
e <- rnorm(n)
X <- rnorm(n)
u <- runif(n, -1, 1)
W <- X + u 
b0 <- 1
b1 <- 2
Y <- b0 + X * b1 + e

lm(
    Y ~ W
)
#> 
#> Call:
#> lm(formula = Y ~ W)
#> 
#> Coefficients:
#> (Intercept)            W  
#>       1.040        1.463

2.3 同時性

これも通常の回帰分析では正しく推定することは出来ない。

3 因果分析としての回帰分析

  • 関心がある変数と誤差項の独立性が重要
  • 関心がある変数と影響がある変数を制御しきれないとき制御が必要
  • 差に注目するなどして変数を制御することで緩和を図る

4 練習問題

説明変数が内生変数であるときのバイアスは、説明変数と誤差項の相関が強いほど大きくなる。 ここでは、数値的に検証する。次は\(\lambda\)を使って、誤差項との相関を制御している。

Code 
n <- 200
b0 <- 1
b1 <- 2

Estimate <- function(lambda) {
    e <- rnorm(n)
    X <- (1 + lambda * e) * runif(n)
    Y <- b0 + X * b1 + e
    
    lm(Y ~ X)$coefficient
}

lambdas <- 0:60 / 100
results <- map(lambdas, \(k) replicate(n, {Estimate(k)}))

真値から補正して、平均なバイアスを見てみる。

Code 
mod_results <- 
    results |> 
    map_dfr(rowMeans) |> 
    mutate(
        `(Intercept)` = `(Intercept)` - b0, 
        X = X - b1
    ) |> 
    mutate(lambda = lambdas)

mod_results |> 
    ggplot(aes(lambda, `(Intercept)`)) + 
    geom_point()
Code 
mod_results |> 
    ggplot(aes(lambda, X)) + 
    geom_point()

上記の結果から、相関が強いほどバイアスが強くなるという傾向が確認できた。

# 外生変数

$$
\begin{align}
Y_i &= \beta_0 + \beta_1 X_i + \epsilon_i \\
i &= 1, ..., n
\end{align}
$$
上記のモデルを考えたとき$X_iが外生変数である$といった意味は、次式で表されるように誤差項と相関しない、
ということを意味する。


$$

\textrm{E}[X_i\epsilon_i] = \textrm{Cov}(X_i, \epsilon_i) = 0

$$

一般に個体$i$の要因$X_i$を分析者側がランダムに決定する場合, $X_i$は明らかに外生変数となる。

#　内生変数

外生変数の条件が満たされない次の状態を考える。

$$

\textrm{E}[X_i\epsilon_i] = \textrm{Cov}(X_i, \epsilon_i) \neq 0

$$


内生変数はいいといして、どのようなケースで内生変数が生じやすいのかを考える。


## 欠落変数

もとのモデルは内生変数がない次の状態を考える。この状態であれば通常の回帰分析で推定することができる。


$$

Y_i = \beta_0 + \beta_1X_{1i} + \beta_2X_{2i} + \epsilon_i

$$

ここで$X_2$が観測できない状態を考える。

$$
\begin{align}

Y_i &= \alpha_0 + \beta_1X_{1i} + \eta_i \\
\alpha_0 &= \beta_0 + \beta_2\textrm{E}[X_{2i}] \\
\eta_i &= \beta_2(X_{2i}-\textrm{E}[X_{2i}])+\epsilon_i


\end{align}
$$

単回帰モデルで正しく$\beta$を推定するためには、$X_{i}$が誤差項と外生的変数であることが求められる。
下記の結果をみると$X_{1i}$は外生的であるため欠落したままでは回帰分析は適していない

$$
\begin{align}
\textrm{E}(X_{1i}) &= \textrm{E}\{X_{1i}[\beta_2(X_{2i}-\textrm{E}[X_{2i}])+\epsilon_i]\} \\
&=\beta_2\textrm{E}{X_{1i}(X_{2i}-\textrm{E}[X_{2i}])} + E[X_{1i}\epsilon_i]\\
&=\beta_2\textrm{Cov}(X_{1i}., X_{2i})
\end{align}
$$
