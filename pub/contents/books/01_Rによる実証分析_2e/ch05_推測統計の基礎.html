3 回帰係数の検定

線形モデルでシミュレーションして回帰係数の値をみてみる。 次の結果からモデル通りの値が出ていることがわかる。

Code 
X <- rnorm(1000, 0, 1)
Y <- 1 + 5 * X + rnorm(1000, 0, 1)
beta1 <- lm(Y ~ X)$coefficients[2]
beta1
#>        X 
#> 4.998162

上記の推定の流れをモンテカルロシミュレーションする。だいたい5に近い値が出ていることがわかる。

Code 
n_simulations <- 1000
betas <- replicate(n_simulations, {
    X <- rnorm(1000, 0, 1)
    Y <- 1 + 5 * X + rnorm(1000, 0, 1)
    lm(Y ~ X)$coefficients[2]
})

hist(betas)
