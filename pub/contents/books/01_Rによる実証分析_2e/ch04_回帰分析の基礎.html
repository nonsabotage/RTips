3 線形回帰

線形回帰モデルは、線形モデルを仮定しているという意味でパラメトリックモデルとなる。

Code 
result <- lm(elec ~ temp, data = tempdata)
result |> 
    tidy()
#> # A tibble: 2 × 5
#>   term        estimate std.error statistic   p.value
#>   <chr>          <dbl>     <dbl>     <dbl>     <dbl>
#> 1 (Intercept)    -614.    143.       -4.29 2.03e-  5
#> 2 temp            145.      5.13     28.2  9.40e-120

重回帰モデルを考える。電力使用量は、時間帯により大きく異なる傾向が見られるので、その情報をつかしたモデルを考える。

Code 
tempdata <- 
    tempdata |> 
    mutate(
        daytime = 1 * (9 <= hour(time) & hour(time) <= 18), 
        elec100 = elec / 100, 
        time12  = hour(time) %% 12, 
        ampm    = ifelse(hour(time) < 12, "a.m.", "p.m.")
    )

fit <- lm(elec ~ temp + daytime + prec, data = tempdata)

日曜日の情報などを追加していく。

Code 
tempdata <- 
    tempdata |> 
    mutate(
        sunday = 1 * (wday(date) == 1), 
        #お盆休み期間
        recess = 1 * (ymd("2014/8/11") <= date & date <= ymd("2014/8/16"))
    )


fit <- lm(
    elec ~ temp + daytime + prec + sunday + recess, data = tempdata
)

fit
