2.1 独立性と相関係数

無相関であるとういことは、独立を意味しないことの例を示す。

Code 
x <- rnorm(100000, 50, 10)
z <- - ((x - 50) ** 2) / 10
cor(x, z)
#> [1] 0.00199626
Code 
plot(x, z)

2.2 確率密度

確率変数\(X\)がある値を取る確率はゼロになってしまう。そこで、ある値に収束するように、確率関数を微分した確率密度関数を使う。

2.3 データによる条件付き期待値の推定

1976年にアメリカ合衆国の男性労働者を対象に実施された賃金調査結果データの1部分を扱う。educが教育年数であり、experは職業経験根数、wageは1時間あたり賃金です。

Code 
path <- here(cur_dir, "data/R_EmpiricalAnalysis_csv/chap03/wage.csv")
wage <- read_csv(path, show_col_types = FALSE)

wage |> 
    head() |> 
    paged_table()
Code 
summary(wage)
#>       educ           exper             wage       
#>  Min.   : 1.00   Min.   : 0.000   Min.   : 100.0  
#>  1st Qu.:12.00   1st Qu.: 6.000   1st Qu.: 394.2  
#>  Median :13.00   Median : 8.000   Median : 537.5  
#>  Mean   :13.26   Mean   : 8.856   Mean   : 577.3  
#>  3rd Qu.:16.00   3rd Qu.:11.000   3rd Qu.: 708.8  
#>  Max.   :18.00   Max.   :23.000   Max.   :2404.0
Code 
plot(wage)

このデータに対して、educ = 12の個人だけを対象にサマリーする。とおもったけど面倒であるので、ここではすべてを対象にする。 これを見るだけでも、どうやら教育年数により収入の平均値が変化していそうなことがわかる。

Code 
wage_summary <- 
    wage |> 
    group_by(educ) |> 
    skim() |> 
    as_tibble() |> 
    filter(skim_variable == "wage")

wage_summary |> paged_table()
Code 
wage_summary |> 
    ggplot(aes(x = educ, y = numeric.mean)) +
    geom_linerange(aes(ymin = numeric.p25, ymax = numeric.p75), lwd = 2) + 
    geom_point(color = "red", size = 5)
