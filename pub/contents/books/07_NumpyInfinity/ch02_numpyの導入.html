4 NumPy配列のインデックスによるアクセス

Code 
x = np.ones(5)
print(x)
[1. 1. 1. 1. 1.]
Code 
x[2] = 8
x
array([1., 1., 8., 1., 1.])

スライスを使った直接の代入も可能である．

Code 
x[:2] = 10
x
array([10., 10.,  8.,  1.,  1.])

リストのスライスを使った代入は，要素数を変える黒魔術であることに注意．

基本的には使わないこと.

Code 
x = list(range(5))
x[1:4] = [-1, -1]
x
[0, -1, -1, 4]

numpyの配列はインプレイスな演算もサポートしている.

Code 
x = np.arange(5)
x[:4] += 10
x[:2] *= 10
x
array([100, 110,  12,  13,   4])

6 ベストプラクティス

上記のようにNumPy配列はループを使わない処理が可能である． 数値演算を高速化する場合には，ループ（ピュアなPython）に依存しないようにすることが重要.

Code 
from mypy.mymodule import TimeExecute
Code 
x = list(range(int(1e7)))
with TimeExecute():
    for i in range(len(x)):
        x[i] *= 2
    
1,260.662 [msec]
Code 
x = list(range(int(1e7)))
with TimeExecute():
    x * 2
262.328 [msec]

9 ブーリアンマスク

NumPyの配列のスライスは，インデックスだけでなく，ブール型でも可能である.

Code 
x = np.arange(5)
is_even = x % 2 == 0
x[is_even] += 10
x
array([10,  1, 12,  3, 14])
