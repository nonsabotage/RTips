3 リストの要素の追加・削除とパフォーマンス

  • Pythonのリストは連結リストのイメージなので末尾への要素追加はコストが小さい
  • 末尾へはappend，任意の箇所へはinsert
Code 
anko_list = ["つぶあん", "こしあん"]
anko_list.append("しろあん")
print(anko_list)
['つぶあん', 'こしあん', 'しろあん']
Code 
anko_list.insert(1, "かぼちゃあん")
print(anko_list)
['つぶあん', 'かぼちゃあん', 'こしあん', 'しろあん']
