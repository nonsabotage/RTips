1 Pyenv

pyenvはpythonのバージョンを切り換えるためのツールである。

windows環境をサポートしていないので、pyenv-winを利用すること。WSLもサポートしていないとのことである。

1.1 インストール

powershellを管理者権限で実行する必要がある。

最新版のバージョン７で実行するとインストールが成功した。

実行が成功すると次のコマンドでpyenvのバージョンを判定する。

pyenv --version

うまくインストールが成功しないときには、実行ポリシーの影響が考えられる。

その際にはこのページを参照する。

1.2 pythonのインストール

次のコマンドでインストール可能なpythonのバージョンを確認する。

pyenv install -l

次のコマンドでpythonのバージョンをインストールする。

pyenv install 3.9.7

いまつかるpythonのバージョンを確認する。

pyenv versions

次のコマンドでグローバルのpythonのバージョンを設定する。

pyenv global 3.9.7

次のコマンドでpythonのバージョンを確認する。

python --version

次のコマンドでローカルのpythonのバージョンを設定する。

pyenv local 3.9.7

次のコマンドでローカルのpythonのバージョンを確認する。

python --version

ローカルを設定すると、そのディレクトリ以下のpythonのバージョンが設定される。

1.3 注意点：パスの設定

pyenv-winを利用すると、pythonのバージョンを切り替えることができる。

もしpythonがシステム環境変数に含まれている場合には次のパスをそれらのパスよりも優先させる必要がある。

%USERPROFILE%\.pyenv\pyenv-win\bin
%USERPROFILE%\.pyenv\pyenv-win\shims

1.4 アンインストール

パスを削除する。パスのフォルダを丸ごと削除する、でよいらしい。

2 Poetry

poetry

2.1 インストール

次のコマンドで公式インストーラーからインストールが可能である。

(Invoke-WebRequest -Uri https://install.python-poetry.org -UseBasicParsing).Content | python -

あとは次のフォルダのパスを通すことでよい。

アンインストールは次のコマンドでアル。

curl -sSL https://install.python-poetry.org | python3 - --uninstall

2.2 プジェクト

新規プロジェクトの場合と、既存のプロジェクトの場合がある。

poetry new my_project

パッケージ開発でないときには次のコマンドで設定は適当な回答で大丈夫である。

cd my_project
poetry init

poetryは２つのプロジェクトモードがある。１つはパッケージモードで、もうひとつはnon package modeである。

2.3 起動

プロジェクトを立ち上げたら基本的には準備よいので、次のコマンドでプロジェクトを立ち上げる。 そうすると設定したプロジェクトが立ち上がる。このとき指定したpythonのインタプリタがインストールされている必要があるので、 事前にpyenvでインストールしておく必要がある。

poetry shell

2.4 その他

2.4.1 依存関係のインストール

poetry install

パッケージのインストールは次のようにおこなう。

poetry add numpy

デフォルトだと仮想環境はアプリケーションのディレクトリに作成されるため、プロジェクト直下に作成するには次のようにする。

poetry config virtualenvs.in-project true

次のように設定すると任意のディレクトリに仮想環境を作成することができる。

poetry config virtualenvs.path <path>

設定したconfigは次のコマンドで確認できる。

poetry config --list

仮想環境を削除するには次のコマンドを実行する。

poetry env list 仮想環境を調べる
poetry env remove <仮想環境パスまたはPythonバージョン>

poetryは通常の依存関係と開発用の依存関係を区別している。開発用の依存関係をインストールするには次のコマンドを実行する。

poetry add --dev pytest

開発用の依存関係をインストールするには次のコマンドを実行する。

poetry install --with dev

開発用の依存関係を除いてインストールするには次のようにする

poetry install --no-dev

開発作業では次のコマンドでリンターなどを実行する。

poetry run pytest
poetry run flake8

プロジェクトの情報を確認するには次のコマンドを実行する。

poetry show
