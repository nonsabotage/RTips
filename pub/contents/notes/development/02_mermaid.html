# Mermaid

- mermadiのサンプルです

# サンプル

## TD

```{mermaid}
graph TD;
    A-->B;
    A-->C;
    B-->D;
    C-->D;
```


```{mermaid}
graph TD;
    %% 入力データと初期値
    A("○から") --> B["資格"] --> C(("英語")) ==> D{"ダイヤ"};
```


## BT

```{mermaid}
graph BT
  A[どこでも誰とでも働ける]
  B[他者に貢献する] --> A
  C[得意なことをギブする] --> B
  D[得意なことを作る] --> C
  E[何が好きかを知る] --> D
  F[説明可能にする] --> D

  G[他者から愛される] --> A
  H["他者との共通点・相違点を見つける"] --> G
  I[好き嫌いをオープンにする] --> H
  E --> I
  J[引き出しを増やす] --> H

  K[自立する] --> A
  L[自分の決めたルールを守る] --> K
  M[依存心をなくす] --> K

  classDef Title font-weight:bold
  class A Title
```


## LR

```{mermaid}
graph LR
    A --> B
    A --> C
    subgraph same_level
      direction LR
      B & C
    end

```

## ER

```{mermaid}
erDiagram
    users ||--o{ tweets : ""

    users {
        bigint id PK "ID"
        varchar name "名称"
        varchar username "ユーザー名"
        varchar description "説明"
        timestamp deleted_at "削除日時"
        timestamp created_at "作成日時"
        timestamp updated_at "更新日時"
    }

    tweets {
        bigint id PK "ID"
        bigint author_id FK "オーサーID:users.id"
        varchar tweet "ツイート"
        timestamp deleted_at "削除日時"
        timestamp created_at "作成日時"
        timestamp updated_at "更新日時"
    }
```

## サブグラフ

```{mermaid}
graph TD;
    subgraph サブグラフ1
        A1[ノードA1]
        A2[ノードA2]
    end
    subgraph サブグラフ2
        B1[ノードB1]
        B2[ノードB2]
    end
    A1 --> B1
```

## シーケンス図

```{mermaid}
sequenceDiagram
    participant A as ユーザー
    participant B as システム
    A->>B: リクエスト送信
    B-->>A: レスポンス返送

```

## ガントチャート

```{mermaid}
gantt
    title プロジェクト計画
    dateFormat  YYYY-MM-DD
    section 準備
        タスク1   :done,    des1, 2024-01-01,2024-01-05
        タスク2   :active,  des2, 2024-01-06, 3d
    section 実行
        タスク3   :         des3, 2024-01-09, 5d
        タスク4   :         des4, after des3, 5d
    section 検証
        タスク5   :         des5, after des4, 3d
        タスク6   :         des6, after des5, 5d
    section リリース
        タスク7   :         des7, after des6, 3d

```
