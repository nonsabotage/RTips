1 hereでrenvはどのように指定すればいいか

  • プロジェクトファイルがあるフォルダで指定するときはそのフォルダになる
  • プロジェクトファイルがないフォルダで指定するときはルートフォルダになる
  • 正確には１番近い親フォルダになっているみたい
  • なのでこの場合のhereの挙動には注意する
Code 
here::here()
#> [1] "H:/Dropbox/R/Workspace/RTipsSite"
cur_dir <- here::here("contents", "test", "R")
print(cur_dir)
#> [1] "H:/Dropbox/R/Workspace/RTipsSite/contents/test/R"
# renv::init(cur_dir)

print(getwd())
#> [1] "H:/Dropbox/R/Workspace/RTipsSite/contents/test/R2"
print(here::here())
#> [1] "H:/Dropbox/R/Workspace/RTipsSite"

# hereでrenvはどのように指定すればいいか

- プロジェクトファイルがあるフォルダで指定するときはそのフォルダになる
- プロジェクトファイルがないフォルダで指定するときはルートフォルダになる
- 正確には１番近い親フォルダになっているみたい
- なのでこの場合の`here`の挙動には注意する

```{r}
#| warning: false
#| message: false
here::here()
cur_dir <- here::here("contents", "test", "R")
print(cur_dir)
# renv::init(cur_dir)

print(getwd())
print(here::here())
```


```{r}
#| warning: false
#| message: false
library(here)
```
