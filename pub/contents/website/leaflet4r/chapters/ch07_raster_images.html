1 Large Raster

terra::resampleを使って解像度を調整すること.

2 Projection Performance

大量のデータがある場合にはleaflet::projectRasterFroLeaflet を使って事前にデータを変換しておくのがよい.

その上でaddRasterImageでproject=FALSEにしておく.

3 Coloring

色々できるという感じです.

4 Example

Code 
r <- rast(here(data_dir, "FG-GML-5339-05-DEM5A.tif")) 

pal <- 
    colorNumeric(
        c("#0C2C84", "#41B6C4", "#FFFFCC"), 
        values(r),
        na.color = "transparent")

leaflet() |> 
    addTiles() |>
    addRasterImage(r, colors = pal, opacity = .5) |> 
    addLegend(
        pal = pal, 
        values = values(r), 
        title = "Surface terrain"
    )
